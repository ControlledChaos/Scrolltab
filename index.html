Scrolltab by Chris Saylor

jQuery plugin to add tabs visually associated to their position relative to the scroll bar.

Examples

// Enables a pin with the default classname
$('p').scrolltab();

// Changes the classname of the created (or existing) pin
$('p').scrolltab({ classname: 'test' });
    
// Change the mouse over event:
$('p').scrolltab({ hoverin: function() { console.log('hovered!'); }});

Demo

View a demonstration of Scrolltab's dynamic behavior.

Options

Display
classname - Name of the class you want the pin to acquire
title - Text to display in the pin
Events
hoverin - function to run on mouse over of the pin
hoverout - function to run on mouse out of the pin
click - function to run on click of the pin

Dependencies

jQuery 1.4+
