Scrolltab by cjsaylor

jQuery plugin to add tabs visually associated to their position relative to the scroll bar.

Examples

  	// Enables a pin with the default classname
	$('').scrolltab();

	// Changes the classname of the created (or existing) pin
	$('').scrolltab({ classname: 'test' });

Example for this page:

 $('h2')
  .click(function() {$(this).slideUp(); })
  .each(function(index) {
    $this = $(this);
    $this.scrolltab({
      hoverin: function() { 
        $(this).stop().animate({width: '200px'}, 500);
      }, 
      hoverout: function() { 
        $(this).stop().animate({width: '100px'}, 500);
      },
      classname: 'scrolltab-item',
      title: $this.attr('title')
    });
  });
